# Simulations 

## Microburst Fan-in → Queue Fill/Drain (Simulation)

This is a single-file HTML/Canvas simulation that illustrates microbursts caused by synchronized senders converging on a bottleneck/queue.

## Run
Open `microburst_queue_sim.html` in a browser:
- macOS Finder: double-click
- or from terminal:
  open microburst_queue_sim.html

## Suggested “Microburst” Settings (for recording)
- Mode: Synchronized
- Senders: 10–12
- Arrival Rate: ~1.2
- Drain Capacity: ~0.8–0.9
- Burst Tightness: ~0.10–0.16
